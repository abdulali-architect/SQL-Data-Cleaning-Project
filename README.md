# SQL-Data-Cleaning-Project
SQL data analysis project demonstrating joins, CTEs, window functions, and analytical problem-solving using real-world data.
## Project Overview
This project demonstrates practical SQL skills through real-world data analysis tasks completed as part of Alex The Analyst’s SQL project series. The focus is on writing clean, efficient queries to extract insights from structured data.

## Tools & Technologies
- SQL (MySQL / PostgreSQL)
- Git & GitHub

## Key Skills Demonstrated
- Data cleaning and transformation
- Joins (INNER, LEFT)
- Subqueries and CTEs
- Aggregate functions
- Window functions
- Analytical problem-solving

## Project Description
The dataset was analyzed to answer business-relevant questions using SQL. Queries were written with readability and performance in mind, following industry-standard practices.

- Converted raw data into actionable insights
